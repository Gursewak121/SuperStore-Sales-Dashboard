# SuperStore-Sales-Dashboard (Power BI)
Power BI Dashboard analyzing SuperStore sales data with interactive visualizations and insights.

## 🌐 Project Overview
This project is a Power BI dashboard created to analyze SuperStore sales data. The dataset was provided in Excel and CSV format. The dashboard provides insights into sales, profit, quantity, and customer behavior using interactive visualizations.

## 🎯 Objectives
 - Analyze overall sales performance.
- Track profit and quantity.
- Compare sales across categories and regions.
- Analyze customer segments.
- Understand payment and shipping trends.
- Forecast future sales.   

## 🛠 Tools & Technologies
- Power BI Desktop
- Power Query
- Data Visualization
## 📁 Dataset
- Source: [SuperStore Dataset](https://github.com/Gursewak121/SuperStore-Sales-Dashboard/blob/main/SuperStore_Sales_Dataset.csv)
- Data includes:   Orders,    Sales,    Profit,    Quantity,    Category,    Region,    Payment Mode,    Ship Mode.

## 📈 Dashboard Features
### KPIs
- Total Sales: 1.7M
- Total Profit: 177.29K
- Total Quantity: 22K
### Visualizations
- Sales by Category.
- Sales by Sub-Category. 
- Sales by Segment.
- Sales by Region.
- Sales by Payment Mode.
- Sales by Ship Mode.
- Monthly Sales Trend.
- Monthly Profit Trend.
- Sales Forecast (15 Days).

## 📊 Insights
- Technology and Office Supplies generate high sales.
- Consumer segment has the highest contribution.
- COD is the most used payment mode.
- Standard Class is the most used shipping mode.
- Sales show consistent growth over time.

## ⚡ How to Use
- Download the [.pbix file](https://github.com/Gursewak121/SuperStore-Sales-Dashboard/blob/main/SuperStoreDashboard.pbix) .
- Open in Power BI Desktop.
- Load the dataset if needed.
- Interact with filters and visuals.
